# NCNU Data Structure and Algorithm Demo II(1102)

Requirements

* GLFW 3
* CMake
* make

## Run this project

This Project includes `dear-imgui` and builds with `CMake`
To Run this project cloning this repo and type following commands.

```bash
$ mkdir build && cd build
$ cmake ..
$ make
```
